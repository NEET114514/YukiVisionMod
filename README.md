# Yuki Vision MOD

妹居物语桌宠全模态 MOD。

## 下载与安装

1. 下载仓库里的 `YukiVisionMod_Package.zip`。
2. 解压后关闭游戏。
3. 双击 `install.bat` 安装。
4. 打开游戏，点左下角“桌宠设置”。
5. 出问题时双击 `uninstall.bat` 恢复。

## 使用前先读

仓库根目录里放了两份 Word 文档，一进仓库就能看到。

请先完整读一遍 [`操作指南.docx`](操作指南.docx)，了解 MOD 的安装、设置和常见问题。

如果你要使用豆包 RTC 桥，再打开 [`豆包RTC桥火山引擎开通指南.docx`](豆包RTC桥火山引擎开通指南.docx)，按里面的顺序开通火山引擎相关服务并填写凭证。
