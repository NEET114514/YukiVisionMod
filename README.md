# Yuki Vision MOD

妹居物语桌宠全模态 MOD。

## 下载与安装

1. 下载仓库里的 `YukiVisionMod_Package.zip`。
2. 解压后关闭游戏。
3. 双击 `install.bat` 安装。
4. 打开游戏，点左下角“桌宠设置”。
5. 出问题时双击 `uninstall.bat` 恢复。

## 主要功能

- HTTP 多模态桌宠：截图拼图、OpenAI 兼容接口、自定义端点、游戏内置 TTS。
- Qwen Realtime：WebSocket realtime 通道、右 Alt 按住说话、自动观察。
- 豆包 RTC：用户自填火山/RTC/ASR/方舟凭证的实验通道。
